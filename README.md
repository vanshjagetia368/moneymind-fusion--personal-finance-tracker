# moneymind-fusion--personal-finance-tracker
A production-grade personal finance management dashboard built using vanilla JavaScript, HTML5, and CSS3. Implements centralized state management, dynamic DOM rendering, persistent storage via localStorage, category-based budgeting, multi-account tracking, real-time financial analytics, CSV export, and a fully responsive FinTech-style UI.
---
## 🔄 Application Flow
1. User performs an action (add transaction, set budget, add category)
2. Application state (`AppState`) is updated
3. State is persisted using `localStorage`
4. `refreshUI()` re-renders the active view
5. UI updates instantly without page reload
